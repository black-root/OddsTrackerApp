Track Odds Application

This application is in a version Beta.

Track the soccer odds second by second in live, or whatever interval time that you want. It provides the information from the page Bet365.com by BetsApi.
This application has to ways to tracks odds:

  • In-PlayEvent: It tracks the odds of the matches already playing in the moment.
  • Upcoming Event: It tracks the odds of the upcoming events, normally you can see matches about the same day that you try searching a match

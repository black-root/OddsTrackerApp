Documentation

This aplication just provides all the information from BetsApi.com, If this website has problems, this aplication will have problems too. BetsApi is only a intermediator from the website Bet365.com

Recomendation: if you want to comparate the data with this aplication, you must go to visit these websites.
(*)This aplicaction is in Beta version, it mean, It could have mistakes.


In-Play Event: it has 4 controls
  • Select League: it has all the leagues playing in this moment
  • Select In-PlayMatch: These matches belong the league already selected in the control "Select League"
  • Tracking Interval: the function of this control is call data time by time, for example: second by second
  • Request Limit: it will be the rows limit of the excel file, it mean the numbers of times to capture the data from the website BET365
